Запуск файла с программой: 
1) Откройте эмулятор терминал;
2) С помощью терминала перейдите в каталог, где лежат файлы: data.txt и PFA.py;
3) Пропишите команду: sudo chmod +x PFA.py;
4) Введите пароль;
5) Затем пропишите команду: python ./PFA.py;
6) Для последующего запуска программы используйте программу из п.6.
